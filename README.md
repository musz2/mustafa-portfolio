# Syed Mustafa Ali — 3D DevOps Portfolio

A premium portfolio based on the original 3D experience, preserved with the same layout, animation system, character scene, cursor behavior, work carousel, and tech-stack physics — updated with Syed Mustafa Ali's DevOps-focused content.

## Highlights

- Original 3D portfolio layout and animation flow preserved
- GSAP-powered transitions and smooth storytelling
- React Three Fiber character scene and physics-based tech stack
- Clean DevOps work case studies with architecture-style visuals
- Updated hero, about, career, and contact content for recruiter-facing use

## Core Stack

- React 18
- TypeScript
- Vite
- GSAP + ScrollSmoother
- Three.js
- React Three Fiber
- React Three Rapier
- React Icons

## Run Locally

```bash
npm install
npm run dev
```

## Production Build

```bash
npm run build
npm run preview
```

## Deployment

Recommended settings for Vercel / Netlify:

- **Framework**: Vite
- **Install Command**: `npm install`
- **Build Command**: `npm run build`
- **Output Directory**: `dist`

## Note

This project intentionally remains a Vite React build in order to preserve the original 3D animation behavior and layout consistency.
